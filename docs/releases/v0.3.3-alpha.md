## Commit inclus

### fix: BUG-045 SkillExecuteResponse n'a pas file_path (ddbeda9)

- **Type** : hotfix critique
- **Fichier** : `src/backend/app/routers/skills.py`
- **Problème** : le code BUG-043 (retry document vide, ajouté en v0.3.2) accédait à `result.file_path` sur un `SkillExecuteResponse` qui n'a pas cet attribut. Crash systématique à chaque génération de document.
- **Fix** : remplacé par `registry.get_file(result.file_id)` qui renvoie le `SkillResult` interne (qui a bien `file_path`).
- **Impact** : tous les utilisateurs qui tentaient de générer un DOCX/PPTX/XLSX avaient une erreur 500.
