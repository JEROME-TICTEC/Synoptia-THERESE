### Bugs corriges (reportes par Dr_logic-3D le 30/03/2026)

**Bug 1 - Choix modele LLM pour agents Atelier**
- Les 6 agents metier de l'Atelier utilisaient Claude Sonnet par defaut, payant sur OpenRouter
- Ajout d'un selecteur de modele LLM dans le catalogue agents (dropdown groupe par provider)
- Le modele selectionne est persiste en localStorage
- Backend : SpawnAgentRequest accepte un `model` override, fallback sur le modele configure par l'utilisateur
- Le endpoint profiles retourne le modele par defaut de l'utilisateur (pas toujours Claude Sonnet)
- Fichiers : schemas_agents.py, agents.py (router), AgentCatalog.tsx, AgentSession.tsx, AtelierPanel.tsx, agents.ts (API)

**Bug 2 - "Utilisation de clarify..." spam (6 lignes)**
- Les outils swarm internes (clarify, create_spec, explain_change) etaient disponibles dans le pool d'outils des agents profil
- Le LLM les appelait → messages "Utilisation de clarify..." affiches dans le chat
- Fix backend : outils swarm exclus du pool d'outils des agents profil
- Fix frontend : messages tool_use pour ces outils internes filtres dans atelierStore
- Fichiers : agents.py (router), atelierStore.ts

**Bug 3 - Depot Git non trouve**
- `_get_source_path()` ne resolvait pas correctement les paths Windows (tildes, quotes, backslashes)
- Fix : `expanduser()` + `resolve()` + support `.git` comme fichier (submodules git)
- Ajout warning log quand le path configure existe mais `.git` non trouve
- Fichier : agents.py (router)
