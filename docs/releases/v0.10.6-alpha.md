## Analyse technique

### BUG-102 - Click tâche Dashboard

**Root cause** : `TaskItem`, `EventItem`, `InvoiceItem`, `ProspectItem` dans `DashboardToday.tsx` avaient `cursor-pointer` CSS mais aucun `onClick` handler. Le curseur changeait visuellement mais le clic ne faisait rien.

**Fix** : Wrapping de chaque item rendu dans `DashboardCard` avec `onClick={onViewAll}`, qui ouvre le panel correspondant (tâches, calendrier, facturation, CRM).

### BUG-103 - Dropdown modèles LLM agents

**Root cause triple** :
1. **Contrat API cassé** : le backend `GET /api/agents/profiles` retourne un tableau nu `[{...}]`, le frontend attendait `{profiles: [...]}`. `profilesResp.profiles` = `undefined` → TypeError silencieux → `setAvailableModels()` jamais appelé → dropdown invisible.
2. **Pas de filtrage** : `AVAILABLE_MODELS` (18 modèles, 8 providers) retourné tel quel sans vérifier les clés API configurées.
3. **Labels incomplets** : `providerLabels` dans AgentsTab manquait OpenRouter et DeepSeek.

**Fix** :
1. `getAgentProfiles()` retourne `AgentProfile[]` (plus de wrapper). `AgentCatalog.tsx` accède au tableau directement.
2. Endpoint `get_config` filtre par providers ayant une clé API en DB ou env var. Ollama inclus si le binaire est sur PATH. Fallback : tous les modèles si aucun provider configuré (premier lancement).
3. Ajout `openrouter: 'OpenRouter'` et `deepseek: 'DeepSeek'` dans `providerLabels`.

### BUG-104 - Détection git

**Root cause** : `known_paths` dans `_get_source_path()` (agents.py) et `_resolve_agents_dir()` (config.py) contenait `"THERESE V2"` (ancien nom) au lieu de `"Synoptia-THERESE"` (nom actuel du dossier). En mode empaqueté (Tauri + PyInstaller), l'auto-détection via `__file__` ne fonctionne pas, et le fallback known_paths ne trouvait rien.

Bonus : `AgentToolExecutor` dans le endpoint spawn était créé sans `GitService`, rendant `git_status()` et `git_diff()` inopérants même quand le chemin était trouvé.

**Fix** :
1. Ajout `home / "Desktop" / "Dev Synoptia" / "Synoptia-THERESE"` en priorité dans les deux fichiers.
2. `GitService` passé à `AgentToolExecutor` quand `source_path` est disponible.
