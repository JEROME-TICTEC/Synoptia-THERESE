### PR #35 - fix: 2 bugs Windows - strftime POSIX + traffic lights macOS

### PR #36 - fix: v0.6.1 - OpenRouter vide + Groq cloud + Gemini Flash Lite
- **Branche** : `fix/v061-openrouter-groq-label`
- **Type** : fix + ajout modèle
- **Décision** : MERGER
- **Détails** :
  - OpenRouter réponse vide : détection `has_content`, gestion `finish_reason` (length, content_filter), erreurs SSE inline, HTTP 401/402/429 avec messages en français
  - Libellé Groq : "Clé API Groq pour la transcription audio (Whisper)" → "Transcription audio via Groq (whisper-large-v3-turbo, cloud)" (suggestion Dr_logic-3D)
  - Gemini 3.1 Flash Lite : ajouté dans token_tracker ($0.25/$1.50), agents config, routers config (Gemini + OpenRouter)
  - BUG-051 : couleurs explicites (#0B1226/#E6EDF7) sur le select modèle pour Linux/GTK
