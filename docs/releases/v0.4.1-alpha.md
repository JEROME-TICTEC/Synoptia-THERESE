## Détails techniques

### BUG-057 - CSP img-src
Le WebView Tauri bloquait les images servies par le backend sidecar sur `http://127.0.0.1:17293`. Ajout de `http://localhost:*` et `http://127.0.0.1:*` dans la directive `img-src` du CSP.

### BUG-059 - Test SMTP refactorisé
`test_connection()` réécrit avec :
- Test IMAP via `asyncio.wait_for(loop.run_in_executor(...), timeout=15.0)`
- Test SMTP via `aiosmtplib.SMTP` avec timeout 15s
- Retourne un dict `{success, imap_ok, smtp_ok, message}` au lieu de `bool`

### BUG-062 - PATH MCP enrichi
Le sidecar PyInstaller a un PATH minimal qui n'inclut pas les installations Node.js. Ajout de la résolution dynamique de :
- `/usr/local/bin`, `/opt/homebrew/bin`
- `~/.nvm/versions/node/*/bin` (version la plus récente)
- `~/.fnm/node-versions/*/bin`
- `~/.volta/bin`

### F-14 - Ollama model listing
Le endpoint `GET /api/config/llm` retournait une liste vide pour Ollama. Ajout d'un appel à `{base_url}/api/tags` pour lister les modèles installés. Même correction pour `POST /api/config/llm`.
