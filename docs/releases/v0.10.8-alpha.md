## Ce qui change

- **BUG-083 (MCP prérequis)** : nouvel endpoint `GET /api/mcp/presets/check-requirements` qui vérifie la disponibilité de npx/node/python/docker. Bandeau jaune dans le panneau MCP si prérequis manquants. L'installation retourne une erreur 422 avec instructions (nodejs.org, python.org, docker.com) si la commande est absente.

- **BUG-084 (Modèles custom)** : nouveau composant `ModelSelector` extrait de LLMTab. Bouton "Custom" pour saisir un identifiant de modèle libre. Placeholder adapté au provider (Anthropic, OpenAI, OpenRouter). Le modèle personnalisé apparaît dans le sélecteur avec la mention "(personnalisé)".

- **BUG-085 (Email UX)** : validation immédiate du format email (regex) côté frontend et backend. Timeout 30s `asyncio.wait_for` côté backend. Timeout 35s `Promise.race` côté frontend. Messages d'erreur contextuels (auth, connexion, timeout).
