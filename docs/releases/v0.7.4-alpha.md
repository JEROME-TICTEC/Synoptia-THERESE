## Changements fonctionnels

### Nouveau : Detection auto de skills (BUG-093)
- L'IA detecte automatiquement quand l'utilisateur demande un document (PPTX, DOCX, XLSX, etc.)
- Syntaxe explicite `{{action: pptx-pro}}` pour forcer un skill
- Execution automatique du skill apres le streaming → fichier telechargetable
- Fichiers : `intent_detector.py` (nouveau), `chat.py`, `schemas.py`, `ChatInput.tsx`, `chat.ts`

### Nouveau : Workspace tools pour l'IA
- L'IA peut lire/envoyer des mails, chercher dans la boite mail
- L'IA peut consulter et creer des evenements calendrier
- 5 tools : read_emails, send_email, search_emails, list_calendar_events, create_calendar_event
- Fichier : `workspace_tools.py` (nouveau), `chat.py`

### Fix : Point decimal pave numerique (BUG-091 complement)
- Les champs prix/quantite acceptent point ET virgule
- Validation client avant envoi (valeurs vides, negatives)
- Ajout aria-label pour accessibilite
- Tests vitest ajoutes

### Fix : Generation PDF facture (BUG-092 complement)
- PDF generable meme sans profil utilisateur configure
- Fix bug pre-existant : .get() sur dataclass → .to_dict()
- Test pytest ajoute
