# Release v0.5.0-alpha - Agents IA Embarqués (Atelier)

**Date** : 5 mars 2026
**Version** : 0.4.8 → 0.5.0
**Type** : Feature majeure (MINOR bump)

## Résumé

Première version de l'Atelier : deux agents IA (Thérèse bot et Zézette bot) embarqués directement dans l'app. Les utilisateurs peuvent demander des améliorations et les agents implémentent les changements sur des branches git avec review obligatoire.

## Contenu

### Backend (14 nouveaux fichiers)
- **Runtime agent** embarqué (format compatible OpenClaw : agent.json + SOUL.md)
- **Swarm orchestrateur** 5 phases (spec → analyse → implémentation → tests → review)
- **GitService** async (16 opérations git via subprocess)
- **13 outils** avec BranchGuard (sécurité écriture branche agent/ uniquement)
- **10 endpoints API** `/api/agents` (SSE streaming, CRUD tâches, approve/reject)
- **3 nouvelles tables** SQLite (agent_tasks, agent_messages, code_changes)
- **34 tests unitaires** (config, bus, tools, git, entities, schemas, swarm)

### Frontend (8 nouveaux fichiers)
- **Panneau Atelier** coulissant (chat agents, mission timeline, code review)
- **Store Zustand** (atelierStore) avec traitement SSE centralisé
- **Diff viewer** simplifié ("Appliquer les changements" / "Refuser")
- **Onglet Agents** dans les paramètres (config, statut, chemin source)
- **Raccourci** Cmd+Shift+A pour ouvrir l'Atelier
- **Bouton Zap** dans le header de navigation

### Open Source
- Configs agents (SOUL.md + agent.json) livrées avec le code
- Override utilisateur possible (~/.therese/agents/{id}/SOUL.md)
- BYOK (Bring Your Own Key) : l'utilisateur fournit sa clé API
- Modèle recommandé : Sonnet 4.6
